# TrainingTracker (FullStack)

Оффлайн помощник для тренировок и поддержки здорового образа жизни.

## Что это?

TrainingTracker - современное Progressive Web App для отслеживания тренировок, которое работает полностью оффлайн 7 дней, генерирует персональные планы на основе твоих данных и напоминает о пропущенных занятиях через push-уведомления.

## Основной функционал (MVP)

### 1. Оффлайн-режим

- Запись всех тренировок без интернетa.
- Подробная статистика: подходы, повторения, веса, время.
- Автоматическая синхронизация при подключении к сети.

### 2. AI персональный план тренеровок

- Анализ прошлых тренировок (уровень, прогресс, пропуски).
- Генерация плана на неделю.
- Адаптация в реальном времени.

### 3. Push-уведомления

- "Дмитрий, Вы не тренеровались уже 3 дня!"
- "Сегодня новый прогресс! Жим +10кг!"
- "Не забывайте о здоровом сне! ❤️"
